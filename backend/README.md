# Recipe Recommendation System (Backend)

## Setup
1) Create MySQL database:
- Open `src/utils/schema.sql` and run it in MySQL (recommended).
- Or just run seed script (it attempts schema creation).

2) Configure `.env`:
Update DB credentials.

## Install & Run
```bash
cd backend
npm install
npm run seed
npm run dev
```
